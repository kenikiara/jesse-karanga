**Live:** https://kenikiara.github.io/jesse-karanga/

- Zero-build static site — HTML + CSS + vanilla JS
- GSAP 3 + ScrollTrigger animations, Lenis smooth scroll (CDN)

- Fully responsive (mobile burger menu, fluid type)
- Real built work from [architectjesse.co.ke](https://architectjesse.co.ke); selected CV projects visualised with AI-generated renders (marked "visualisation")
